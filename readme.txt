Это приложение WinForms для службы доставки, которое фильтрует заказы в зависимости от количества обращений в конкретном районе города и времени обращения с и по.

Для запуска приложения необходимо импортировать данные файлы в папку и открыть ее в Visual Studio. Необходимые пакеты установлены в папке packages: Newston.Json.13.0.3

Убедитесь, что у вас установлен .NET Framework 4.7.2 или выше и установлены все nuget пакеты. Не забудьте проверить, что у Вас есть права на чтение и запись у файлов, которые Вы тестируете

В папке ..DeliveryApp\bin\Debug хранятся файлы db.json и StandartLoggingPath.txt. Первый используется в качестве данных на вход для нашего приложения. В формате json он хранит id, вес заказа, район и время заказа. Второй файл используется как путь для логов по умолчанию для того, чтобы хранить логи, пока пользователь еще не выбрал свой путь при необходимости.

Все заказы, которые содержит файл db.json были созданы 25 октября 2024 года. При необходимости вы можете изменить/добавить свой файл db.json сохраняя структуру. Рекомендую для теста устанавливать дату 2024-10-25 17:00:00 и район "Восточный". Это один из случаев, когда в выборку попадут сразу несколько заказов.

Приложение было покрыто тестами, а файл Logging.cs используется для логирования.
